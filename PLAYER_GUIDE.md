# Letisland 玩家使用指南

欢迎来到 Letisland 服务器！本指南面向**普通玩家**，介绍游戏内可用的所有功能模块以及对应指令。管理员专属命令会在每个模块末尾单独标注。

> 提示：所有指令均支持 Tab 自动补全。在指令后面加空格后按 Tab 可看到可用子命令。

---

## 功能模块总览

| 模块 | 主命令 | 别名 | 用途 |
|------|--------|------|------------|
| 经济系统 | `/economy` | `/eco`、`/money` | 余额查询、玩家间转账 |
| 商店系统 | `/shop` | `/store` | 定时刷新的商品买卖，标题实时显示余额 |
| 刷石机系统 | `/generator` | `/gen` | 全服共享的刷石机等级，玩家可集资升级 |
| 钓鱼科技系统 | `/fishing` | `/fish` | 钓鱼升级、鱼图鉴、自动出售 |
| 称号系统 | `/title` | `/titles` | 解锁称号并佩戴，展示在聊天名前 |
| 垃圾桶系统 | `/trash` | — | 定时清理世界掉落物，可从 GUI 取回 |

---

## 一、经济系统

主命令 `/economy`，别名 `/eco`、`/money`。货币符号默认为 `$`，新玩家起始余额为 0。

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/economy balance` | 查询自己的余额 |
| `/economy balance <玩家>` | 查询其他玩家余额（需要 `letisland.economy.balance.others` 权限，默认仅 OP） |
| `/economy pay <玩家> <金额>` | 向指定玩家转账（金额必须大于 0，不能给自己转账） |

**示例：**
```
/economy balance
/economy pay Steve 100.5
```

---

## 二、商店系统

主命令 `/shop`，别名 `/store`。打开后会显示一个箱子 GUI：

- **上半部分**为**购买区**：左键点击购买对应商品（扣余额、给物品）
- **下半部分**为**出售区**：左键点击出售背包中对应物品换钱
- **中间分隔栏**：红色玻璃板显示距下次自动刷新的分钟数
- **GUI 标题**实时显示你的当前余额，每次交易后会自动更新

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/shop` 或 `/shop open` | 打开商店 |

### 使用小贴士

- **出售不需要手持物品**：出售区会自动从你的整个背包（含快捷栏）中扣除对应物品，只要背包中数量足够即可。
- **背包空间不足时无法购买**：购买前请保证有空余格子。
- 商店会**定时自动刷新**商品，刷新时全服会收到通知；管理员也可手动刷新。

## 三、刷石机系统

主命令 `/generator`，别名 `/gen`。**全服共享一个刷石机等级**，任何玩家升级后全服都会享受新等级，并广播通知。

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/generator` 或 `/generator info` | 查看当前等级、矿物种类、替换概率，以及下一级信息和升级费用 |
| `/generator upgrade` | 升级刷石机（消耗自己的余额） |

### 使用小贴士

- 升级成功后，全服会在聊天栏收到通知。
- 已达最高等级时 `/generator info` 会提示，并不再显示下一级信息。

---

## 四、钓鱼科技系统

主命令 `/fishing`，别名 `/fish`。钓鱼可获得不同品质的鱼，钓鱼会累计经验并升级，等级越高可钓到的鱼品质越好。

**自动出售说明：**
设置为 N 时，品质等级 ≤ N 的鱼钓上来会**自动换成金币**，无需手动卖。设为 `0` 表示关闭自动出售。

### 使用小贴士

- 品质等级范围是 `1-5`，等级越高越稀有、越值钱。
- 自动出售功能可能被管理员全局关闭；若提示"已被管理员关闭"，请联系管理员。

## 五、称号系统

主命令 `/title`，别名 `/titles`。佩戴称号后，称号会显示在你的**聊天消息前**，作为身份标识。

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/title` 或 `/title codex` | 打开**称号图鉴 GUI**，查看所有称号与解锁方式 |
| `/title list` | 在聊天栏列出所有称号及解锁状态 |
| `/title set <称号ID>` | 佩戴一个已解锁的称号 |
| `/title set none` | 取消当前佩戴的称号 |

**示例：**

```
/title set novice
/title set none
```

### 使用小贴士

- 在图鉴 GUI 中可直观查看每个称号的解锁方式（如达成某条件、解锁某图鉴等）。
- 佩戴后，你的聊天名前会加上对应颜色与样式的称号前缀。
- 想取消就 `/title set none`，不影响已解锁状态。

---

## 六、垃圾桶系统

主命令 `/trash`。系统会每隔 **5 分钟**自动清理全世界的掉落物，并收入"垃圾桶"。玩家可随时打开垃圾桶 GUI 取回被清理的物品。

### 玩家命令

| 命令 | 说明 |
|------|------|
| `/trash` | 打开垃圾桶 GUI |

### 使用小贴士

- 垃圾桶是**可翻页的箱子 GUI**，每页 45 格，可容纳的物品总量**理论无限**。
- 左键点击物品即可**取出**到背包中（背包满时物品会掉在你脚下）。
- **底部导航栏**：左侧为"上一页"，右侧为"下一页"，中间为页码信息。
- 清理掉落物后，全服会收到提示："已清理 X 个掉落物，使用 `/trash` 可取出"。
- 如果你不小心把贵重物品掉在地上，5 分钟内还能在垃圾桶里找回来！

## 常见问题

**Q：新玩家没有余额怎么办？**
A：默认起始余额为 0，可以通过钓鱼出售鱼、出售物品给商店、或其他玩家转账获取。

**Q：商店什么时候刷新？**
A：商店会定时自动刷新，刷新前可在商店 GUI 中间分隔栏看到剩余分钟数；刷新后全服会收到广播通知。

**Q：我掉的物品能找回吗？**
A：地上的掉落物每 5 分钟被清理进垃圾桶，5 分钟内通过 `/trash` 即可取回（建议尽快取回，避免被新清理的物品挤占页面）。

**Q：钓鱼怎么升级？**
A：钓鱼会累计经验，达到当前等级所需经验后自动升级；也可用 `/fishing upgrade` 打开升级 GUI 直接花费余额升级。

**Q：称号怎么解锁？**
A：每个称号有不同解锁条件，可在 `/title` 图鉴 GUI 中查看具体解锁方式。

**Q：指令记不住怎么办？**
A：所有命令均支持 Tab 补全；任意命令后跟 `help` 或直接输入主命令（如 `/shop`）都会显示帮助。
